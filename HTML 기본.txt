<input> 다양한 종류의 입력 데이터 유형과 컨트롤 위젯(HTML에서 제일 강력하고 복잡한요소)
EX) <input type="text"> / <input type="range" min="0" max"10"> 다양한 속성
<input type="radio"> = 여러 선택지중 하나만 선택가능한 버튼 속성(클릭시 취소불가 번복만가능)
EX) 남자:<input type="radio" name="gender">
     여자:<input type="radio" name="gender"> 
<link> 다른 리소스 연결(<head>에 포함)
EX) <link rel="stylesheet" href="style.css">
      style.css파일을 만들어 css코딩한후 적용
rel : 링크된 대상의관계 / href :링크 목적지 / media : 어떤 매체에 적용되는지 나타냄
